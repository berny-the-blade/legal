How to decide

  • Look at the 48px versions first — that's what Android launchers show on the home screen. If it's not readable, it's wrong.
  • Squint at each — if it disappears into a green blob, the contrast is too low. A, B, C should all hold up.
  • Cover the description — does the icon alone tell you it's a domino app?
  • Mental Play Store test — imagine these in a grid next to Don Naipe and Domino Master. Which one would you tap?
